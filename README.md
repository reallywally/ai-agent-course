## 리뷰

- 각 섹션 마다 독립적인 AI Agent를 개발
- 원래 AI Agent를 개발하는게 이런 맥락인지 모르겠지만 강의는 코딩 자체 보다 문제를 해결하는 프롬프트를 많이 알려줌
- 프레임워크로 랭체인, 랭그래프가 아니고 crewai라서 다음 프로젝트 스펙에는 맞지 않음

- 그래서 각 섹션별 최종 코드만 남기기로함
